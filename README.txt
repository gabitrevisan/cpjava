Apresentação
Introdução
O código desenvolvido para este Checkpoint é uma implementação em Java de um programa de criptografia e descriptografia
de frases. Ele oferece duas opções principais: encriptar uma frase e descriptografar uma frase. Para cada opção, o
usuário pode escolher entre dois métodos de criptografia: Cifra de César e Código Morse.


Estrutura do Código
O código está dividido em várias seções, sendo elas:

1. Imports e Declaração da Classe: O programa começa importando a classe Scanner para entrada de dados do usuário e
declara a classe Criptografia.

2. Função main: A função main é o ponto de entrada do programa. Ela exibe um menu para o usuário escolher entre
encriptar ou descriptografar uma frase e chama a função correspondente com base na escolha do usuário, funcionando como
nosso "menu principal".

3. Métodos de Criptografia e Descriptografia: São implementadas quatro funções principais para criptografar e
descriptografar frases utilizando os métodos de Cifra de César e Código Morse.
    - cifraCesar: Criptografa uma frase usando a Cifra de César.
    - decifraCesar: Descriptografa uma frase cifrada usando a Cifra de César.
    - codificaMorse: Criptografa uma frase usando o Código Morse.
    - decifraMorse: Descriptografa uma frase cifrada usando o Código Morse.
    - Comentários no Código: Cada parte do código está acompanhada de comentários explicativos, detalhando sua
    - funcionalidade e lógica de implementação.

4. Funcionalidade Detalhada
    - Menu de Opções: O programa exibe um menu inicial para o usuário escolher entre encriptar ou descriptografar
    uma frase.
    - Escolha do Método de Criptografia: Para cada opção (criptografar ou descriptografar), o usuário pode escolher
    entre dois métodos de criptografia: Cifra de César ou Código Morse.
    - Entrada do Usuário: Após escolher o método de criptografia, o usuário é solicitado a inserir a frase a ser
    encriptada ou descriptografada.
    - Processamento da Frase: A frase inserida pelo usuário é processada de acordo com o método de criptografia
    escolhido.
    - Exibição do Resultado: O programa exibe o resultado da encriptação ou descriptografia da frase.


Considerações Finais
O código apresenta uma implementação funcional e bem estruturada de um programa de criptografia e descriptografia de
frases. Ele oferece opções claras para o usuário e utiliza métodos simples, mas eficazes, de criptografia. Os
comentários fornecidos no código ajudam a entender a lógica por trás de cada parte do programa.
